# Scratch Trainer

This repo is a scratch practice app built with Expo/React Native (plus a simple web demo via `scratch-trainer.html`).

## What it is
A structured scratch-training workflow: quick exercises and full session plans.

## Who it's for
DJs who want consistent practice reps and progression.

## Features
- Difficulty selector: הכל / בסיס / בינוני / מתקדם / מומחה
- Session plan filters exercises by difficulty
- Idle screen shows exercises & timers

## Quick start
1. Clone and install:
   - `bun install` (or `npm install`)
2. Run with Expo:
   - `npx expo start`
3. Web demo:
   - open `scratch-trainer.html` in a browser
